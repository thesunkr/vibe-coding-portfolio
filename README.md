# 누나네 화분가게 🪴

정성을 담아 키운 건강한 식물과 예쁜 화분을 소개하는 포트폴리오 웹사이트입니다.

## 🌿 프로젝트 소개

**누나네 화분가게**는 8년째 동네에서 사랑받는 작은 화분가게의 온라인 포트폴리오입니다.

## ✨ 주요 기능

- 📱 **반응형 디자인** - 데스크톱, 태블릿, 모바일 모두 지원
- 🎨 **따뜻한 디자인** - 자연 친화적인 색상과 친근한 분위기
- 🔐 **로그인 페이지** - 터치 친화적인 모바일 최적화
- 🛒 **상품 카테고리 필터** - 실내식물, 다육식물, 화분 분류
- 📞 **문의 양식** - 고객 문의 기능

## 🛠️ 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - 커스텀 스타일 & 애니메이션
- **Tailwind CSS** - 유틸리티 기반 스타일링
- **JavaScript** - 인터랙티브 기능

## 📁 파일 구조

```
vibe-coding-portfolio/
├── index.html      # 메인 페이지
├── login.html      # 로그인 페이지
├── styles.css      # 커스텀 CSS
├── main.js         # JavaScript
└── README.md       # 프로젝트 설명
```

## 🚀 실행 방법

1. 저장소 클론
```bash
git clone https://github.com/thesunkr/vibe-coding-portfolio.git
```

2. `index.html` 파일을 브라우저에서 열기

또는 Live Server 확장 프로그램 사용

## 📱 반응형 지원

- ✅ 터치 친화적 버튼 (최소 44px)
- ✅ 가로/세로 화면 회전 대응
- ✅ 모바일 키보드 고려 레이아웃
- ✅ Safe Area 지원 (노치 디바이스)
